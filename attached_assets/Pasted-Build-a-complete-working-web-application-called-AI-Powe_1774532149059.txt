Build a complete working web application called:

“AI-Powered Air Quality Intelligence System – India 🇮🇳”

This project must combine:

* Real dataset (from Kaggle CSV)
* Machine Learning model (Linear Regression)
* Interactive web dashboard
* India map-based city selection
* Chemical analysis logic

📁 PROJECT STRUCTURE

Create:

* app.py (Flask backend)
* templates/

  * index.html
  * dashboard.html
  * calculator.html
  * research.html
* static/

  * style.css
  * script.js
* dataset.csv (Kaggle dataset)

---

⚙️ BACKEND (Flask)

1. Load dataset (CSV using pandas)

2. Preprocess data:

* Handle missing values
* Select features:

  * temperature
  * humidity
  * wind speed
* Target:

  * AQI

3. Train ML model:

* Use Linear Regression from sklearn

Example:

* X = [temp, humidity, wind]
* y = AQI

4. Create routes:

/ → Home page (map)
/dashboard/<city> → show city data
/predict → handle AQI prediction (POST request)
/calculator → input page

---

🤖 MACHINE LEARNING

Train model like:

* model.fit(X, y)

Prediction:

* model.predict([temp, humidity, wind])

Return predicted AQI to frontend
