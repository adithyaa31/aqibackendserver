# ML Calculation Demo (Standalone)

This folder contains example ML-style code that **does not connect** to the rest of the project.
It is meant to provide demonstrable ML code for an **AQI calculation**.

## Files
- `calc_ml.py`:
  - `predict_aqi(...)`: ML-style calculation using feature engineering + weights.
  - `train_demo_model(...)`: trains a tiny linear regressor on synthetic data (demo pipeline).

## Run
```bash
python calc_ml.py
```
